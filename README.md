# Spring Boot Multilanguage Project

Projeto de estudo demonstrando uma aplicação **Spring Boot** utilizando múltiplas linguagens da **JVM** no mesmo projeto.

O objetivo é explorar a interoperabilidade entre:

* Java
* Kotlin
* Groovy

Todas rodando na mesma aplicação **Spring Boot**.

---

## 🚀 Tecnologias utilizadas

* Java 17
* Spring Boot
* Maven
* Kotlin
* Groovy
* REST API
* Git / GitHub

---

## 📁 Estrutura do projeto

```
src
 └── main
     ├── java
     │   └── controllers e services principais 
     ├── kotlin
     │   └── models e DTOs
     ├── groovy
     │   └── regras de negócio dinâmicas
     └── resources
```

---

## ▶️ Como executar o projeto

Clone o repositório:

```
git clone https://github.com/SEU_USUARIO/multilang.git
```

Entre na pasta do projeto:

```
cd multilang
```

Execute a aplicação:

```
./mvnw spring-boot:run
```

A aplicação será iniciada em:

```
http://localhost:8080
```

## 🎯 Objetivo do projeto

Este projeto tem como objetivo demonstrar:

* interoperabilidade entre linguagens da JVM
* uso do Spring Boot em arquitetura simples
* estrutura de projeto para APIs REST
* experimentação com Kotlin e Groovy dentro de aplicações Java
